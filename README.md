# Welcome to SyncUp

SyncUp is a student made networking app focused on connecting students and community building.

## Getting Started

### 1. Install Dependencies

```bash
npm install --legacy-peer-deps
```

### 2. Start the Project

- Start the development server (choose your platform):

```bash
npx expo start --clear   # dev mode
npm run build            # web build
```

- Reset the project (clear cache, etc.):

```bash
npm run reset-project
```

### 3. Lint the Code

```bash
npm run lint
```

## Main Dependencies

- React Native: 0.81.5
- React: 19.1.0
- Expo: ~54.0.25
- Supabase: ^2.50.0
